# Rohling-Higgs Symmetry Simulation (v11)

This repository contains the numerical verification for the **Rohling-Higgs Coupling** model.

## Overview
The simulation solves the coupled scalar field equations for the Rohling field ($R$) and the Higgs field ($H$). It demonstrates the stability of the vacuum and the mass splitting that leads to the predicted **58.1 GeV boson**.

## Technical Details
- **Coupling Constant ($eta$):** 0.4774
- **Language:** Python 3.x
- **Dependencies:** NumPy, Matplotlib

## Scientific Context
This code is the technical counterpart to the paper submitted to viXra (February 2026).
- **Author:** [Marc Rohling]
- **Original Concept:** 2024
- **Validation Data:** CERN CMS/ATLAS 2026 (58.1 GeV anomaly)

## Usage
1. Ensure you have Python installed.
2. Install dependencies: `pip install numpy matplotlib`
3. Run the script: `python Rohling_Higgs_Sim.py`

The script will generate the field oscillation patterns and mass distribution charts confirming the theoretical predictions.
